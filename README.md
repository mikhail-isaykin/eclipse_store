# Eclipse Store

## О проекте

Данный репозиторий содержит тестовый Django-проект, созданный в учебных целях
в рамках освоения веб-разработки на Python. На текущем этапе проект представляет
собой экспериментальную площадку для отработки навыков работы с фреймворком Django,
включая построение архитектуры приложения, работу с ORM, настройку маршрутизации,
шаблонизацию и деплой.

## Текущее состояние

Проект находится на стадии активной разработки и обучения. Структура, логика и
кодовая база могут кардинально меняться по мере углубления в технологии.
Не рекомендуется использовать данный код в production-среде — он не проходил
аудит безопасности, не оптимизирован под нагрузку и не покрыт тестами в
достаточной мере.

В рамках текущей версии реализованы базовые механики интернет-магазина:
просмотр каталога товаров, страницы отдельных позиций, административная панель
и простейшая логика отображения контента.

## Планы

В перспективе проект будет полностью переработан и превращён в полноценное
production-ready приложение. Планируется расширение функциональности, рефакторинг
архитектуры, подключение внешних сервисов, оптимизация запросов к базе данных
и настройка CI/CD. Текущая версия служит фундаментом и точкой отсчёта.
